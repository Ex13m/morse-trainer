# История изменений

## v2.0.0 — 2026-05-13

Первый релиз. Полная реализация Morse Trainer на React + TypeScript + Vite.

### Добавлено

- **Рамка устройства** — корпус 414×896 px в стиле латунного радиоприёмника, статусбар с часами/сигналом/батареей, нотч
- **Переключатель языка** RU / EN с мгновенной сменой алфавита, фраз и всех надписей
- **4 режима работы:**
  - **ТРЕНИРОВКА** — случайная фраза из тематического пула (МОРЕ / ВОЙНА / ШПИОН / SOS / МИКС), подсветка следующей ожидаемой буквы в дереве (hint-пульс), зелёный pop при правильном вводе, красный при ошибке
  - **СВОБОДНЫЙ ТЕКСТ** — свободный набор ключом, распознанные буквы накапливаются в поле, pending-индикатор текущей последовательности
  - **РЕЖИМ КОДА** — ввод текста с клавиатуры, кнопка ПЕРЕДАТЬ с пошаговой анимацией дерева и озвучкой, кнопки WAV / ФАЙЛ / МИКРОФОН
  - **НАСТРОЙКИ** — частота тона (300–1200 Гц), порог тире (80–500 мс), скорость (5–40 WPM), вибрация, сброс по умолчанию
- **Дерево Морзе** — SVG-визуализация префиксного дерева, точки влево (красные пунктирные рёбра), тире вправо (жёлтые пунктирные), активная цепочка зелёным, flash-анимация при коммите буквы
- **Латунный ключ** — круглая кнопка 118 px с радиальным градиентом, анимация нажатия (translateY 6px + тени), 4 декоративных винта
- **Threshold meter** — полоса с 20 тиками, зелёно-кремовый градиент заполнения, белый маркер порога, цифровой индикатор мс
- **LED-индикаторы** — КОРОТКО / ДОЛГО загораются зелёным при нажатии ключа
- **Scope bar** — визуализация текущей последовательности точек/тире внизу панели дерева
- **Center letter pop** — большая буква (110 px) с glow-анимацией при распознавании, красный вариант при ошибке в тренировке
- **Кодек-модуль:**
  - WAV export — синтез через OfflineAudioContext, 16-bit PCM, скачивание файла
  - File decode — импорт аудиофайла, envelope-анализ, адаптивный порог, авто-определение алфавита (RU/EN)
  - Mic decode — захват микрофона через getUserMedia, AnalyserNode, адаптивная оценка длины точки, live-буфер morse
- **Аудио** — синусоидальный осциллятор с attack 12 мс / release 20 мс, настраиваемая частота
- **Вибрация** — 20 мс при нажатии ключа, 18/50 мс при передаче точки/тире
- **Spacebar** — глобальная клавиша пробел работает как ключ (кроме полей ввода)
- **Дизайн-токены** — полная латунная палитра (8 ступеней), 4 шрифтовых семейства, все размеры и радиусы из прототипа
