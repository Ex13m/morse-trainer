Master Morse code the way real radio operators did — with a brass telegraph key under your thumb.

MORSE TRAINER is a beautifully crafted trainer with an engraved-brass aesthetic that teaches you to send and receive Morse code in 13 languages.

• Training — key themed phrases (sea, war, spy, SOS, pirates, space, animals) with a live Morse tree that guides every letter
